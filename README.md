Poll Bot is a tool for all discord servers

# Setup

To setup the bots code, You need to follow these steps

1. Create a file called `.env`
2. In the `.env` file, Type this

BOT_TOKEN=

After the equal sign, Put your bots token, Then it should look like this

BOT_TOKEN=Your bots token


If you have any errors with the code, Join our server https://discord.gg/sqqYRr8GdY
